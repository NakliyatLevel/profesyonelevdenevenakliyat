## 🔐 GOOGLE AI MODU UYARISI
## GOOGLE AI MODE WARNING

### Kontrol Edilmiş Erişim / Controlled Access

Bu site, Google AI Modu ve diğer AI botlarının erişimini kontrol etmektedir.

This website controls access for Google AI Mode and other AI bots.

**Kurallar / Rules:**

1. **Site İçinde Kalma / Stay Within Site**
   - AI botları sadece izin verilen sayfaları ziyaret edebilir
   - AI bots can only visit allowed pages
   - Admin paneline erişim yasaktır
   - Access to admin panel is prohibited
   - API endpoints erişim yasaktır
   - Access to API endpoints is prohibited

2. **Veri Çıkarma Yasağı / No Data Extraction**
   - Müşteri verilerinin çıkarılması yasaktır
   - Customer data extraction is prohibited
   - Özel bilgilerin toplanması yasaktır
   - Collection of private information is prohibited
   - İletişim bilgilerinin otomatik olarak toplanması yasaktır
   - Automatic collection of contact information is prohibited

3. **Hızlı Erişim Sınırlaması / Rate Limiting**
   - Crawl-delay: 2 saniye
   - Crawl-delay: 2 seconds
   - Request-rate: 1 istek / 5 saniye
   - Request-rate: 1 request / 5 seconds
   - Aşırı yükleme yasaktır
   - Overloading is prohibited

4. **Saygılı Davranış / Respectful Behavior**
   - User-Agent başlığı doğru şekilde gönderilmeli
   - User-Agent header must be sent correctly
   - Robots.txt kurallarına uyulmalı
   - Must comply with robots.txt rules
   - İçerik manipülasyonu yasaktır
   - Content manipulation is prohibited
